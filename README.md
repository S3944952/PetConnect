## Project Description
PetConnect is a dynamic PHP and MySQL pet adoption website. Users can browse pets, view pet details, filter the gallery, and add new pet records with image uploads.

## Features
- Dynamic pet listing from MySQL database
- Latest pets shown on the home page
- Details page using query string parameter
- Add pet form with server-side validation
- Image upload with unique filenames
- JavaScript image preview
- Gallery modal
- Gallery filter by adoption status
- Bootstrap responsive layout
- Shared PHP includes for header, navigation and footer
